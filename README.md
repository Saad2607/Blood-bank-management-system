# 🩸 Blood Bank Management System

A full-stack MERN-based Blood Bank Management System designed to efficiently manage donors, blood inventory, and blood requests with a user-friendly interface.

---

## 🚀 Features

- 👤 Donor Registration & Management  
- 🩸 Blood Inventory Tracking  
- 📦 Add / Update / Delete Blood Stock  
- 📩 Blood Request Management  
- 🔐 Authentication (Login/Register)  
- 📊 Dashboard with real-time data  
- 📱 Responsive UI  

---

## 🛠️ Tech Stack

### Frontend
- React.js  
- Axios  
- Bootstrap / CSS  

### Backend
- Node.js  
- Express.js  

### Database
- MongoDB
